Introduction
===========

This is the description of the data in the tidy data set. The original data comes from:
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

The data provided for that assessment comes from:
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

Variables
=========
column 1: activity label (factor) by its name (e.g. "WALKINg", "STANDING", ...)

features taken from the original set are:
* tBodyAcc-XYZ (int)
* tGravityAcc-XYZ (int)
* tBodyAccJerk-XYZ (int)
* tBodyGyro-XYZ (int)
* tBodyGyroJerk-XYZ (int)
* tBodyAccMag (int)
* tGravityAccMag (int)
* tBodyAccJerkMag (int)
* tBodyGyroMag (int)
* tBodyGyroJerkMag (int)
* fBodyAcc-XYZ (int)
* fBodyAccJerk-XYZ (int)
* fBodyGyro-XYZ (int)
* fBodyAccMag (int)
* fBodyAccJerkMag (int)
* fBodyGyroMag (int)
* fBodyGyroJerkMag (int)

Summary
=======
The data for mean and standard deviation has been selected. Those columns with mean() and std().
